How to Use SecureCrypt

  1. Select your encryption algorithm (AES-256 or ChaCha20)
  2. Enter a strong password in the Security Key section
  3. Choose whether to encrypt Text or a File
  4. Click Encrypt / Decrypt buttons depending on your need
  5. Copy your encrypted message or download the encrypted file
  6. To view .enc file content, use the Encrypted File Viewer below

All encryption happens locally in your browser — no data ever leaves your device.
